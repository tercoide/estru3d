Bienvenidos a Estru3D version 0.1b, este programa es una herramienta para la resolución de problemas estructurales que utiliza el Método de la Rigidez como base teórica. Le recuerdo que el uso del mismo implica la completa aceptación de los Términos y Condiciones. 

¿Por que versión BETA? Las versiones BETA de software son ediciones que se publican a los efectos de que los usuarios las prueben e informen de los errores que contengan o las sugerencias para la versión comercial. Esto se hace para de lograr un más rápido desarrollo del producto, por lo tanto el autor recomienda no utilizar versiones BETA para trabajo real y sí hacerlo para comprobar trabajos teóricos o comparar otro software.
