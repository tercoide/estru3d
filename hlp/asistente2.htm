Asistente

Desde el menú 'Asistente' se accede a una serie de funciones que permiten generar o modificar el diseño la estructura:

Asistente para generar estructuras

Aqui debería comenzar si es la primera vez que se ejecuta el programa. Se abre una ventana donde se es guiado para diseñar a la estructura de modo que pueda representar a la estructura real que se quiere calcular. Además se hacen verificaciones para evitar errores de diseño para los usuarios noveles.

Optimizar nudos

Esta rutina optimiza la ubicación de los nudos para que el ancho de banda de la matriz sea el menor posible. Por lo general no es necesario utilizar esta rutina, a menos de que el programa indique que no hay memoria suficiente para realizar los cálculos.

Eliminar nudo - barra

Esto permite extraer de las listas un nudo o una barra, con lo que se reacomodan los que quedan. Esto es: si elimino el nudo 12, los datos del 13 pasan al 12, los del 14 al 13 y así. Lo mismo ocurre si elmino una barra.

Eliminar nudos coincidentes

Se eliminarán los nudos que tengan las mismas coordenadas.

Eliminar barras coincidentes

Análogo al proceso anterior, esta rutina opera sobre las barras que tienen iguales nudos inicial y final, con lo que coinciden en el mismo lugar físico. Pero en este caso pude especificarse que es lo que se quiere hacer con los valores de las barras: área, inercias, módulos de elasticidad. Se puede adoptar los de la primer barra, o promediar o sumar.

Generar pórtico ortogonal

Se abrirá una ventana en la que se indican el nudo inicial a partir del cual se procederá a generar un entramado de barras, con sus nudos respectivos. Puede ser un entramado espacial o plano (basta poner un número de tramos igual a cero en un sentido). Se debe indicar el paso que es la distancia entre nudos.
